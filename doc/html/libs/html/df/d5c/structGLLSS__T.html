Detailed Description

The Link Loss server internal structure for the server role.

This structure is visible to the application as it is responsible for managing resource to pass to the Link Loss library. The elements of this structure are only modified by the Link Loss library. The application SHOULD NOT modify the values at any time as it could lead to undefined behaviour.
